# qw
a query word command by golang

## install
````shell
go install
````

## example
````shell
qw apple
英 ['æp(ə)l]   美 ['æpl]
n. 苹果，苹果树，苹果似的东西；[美俚]炸弹，手榴弹，（棒球的）球；[美俚]人，家伙。

1. I twisted an apple off the tree.
我从树上摘下来一个苹果。

2. He hemisected an apple and gave it to me.
他把一个苹果切成了两半，把一半给了我。

3. He nibbled away the apple peel, then ate the flesh.
他先慢慢地咬去苹果皮，然后再吃果肉。
````
